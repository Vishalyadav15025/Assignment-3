Canadian Provinces API Documentation

Introduction

Welcome to the Canadian Provinces API documentation. This API provides information about the provinces of Canada.

Endpoints

The following endpoints are available:

  • GET /provinces: Get information about all Canadian provinces.

Examples

Here are some examples of how to use the API:

        
fetch('/provinces')
  .then(response => response.json())
  .then(data => console.log(data))
  .catch(error => console.error('Error:', error));

Response

The API returns a JSON object with the following structure:

        
{
  "provinces": [
    // Example province object
    {
      "name": "Alberta",
      "abbreviation": "AB",
      "capital": "Edmonton",
      "population": 4421876,
      "image": "https://example.com/alberta.jpg",
      "randomFact": "Alberta is home to the famous Banff National Park, one of the oldest national parks in the world."
    },
    // Add more province objects as needed
  ]
}
